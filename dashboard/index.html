Quick start

Install Noticeify in 3 steps

Works on Shopify, WordPress, or any HTML site.

1

Add the embed tag

Place this as the very first script in <head>, before GTM. On Shopify, add to theme.liquid immediately after <head>.

<script
  src="https://noticeify.pages.dev/consent-loader.js"
  data-client-id="nfy_ocusleep_prod"
  data-domain="ocusleep.com"
  async></script>
2

Gate your vendor scripts

Use the Noticeify API to check consent before loading any pixel or script.

// Load Meta Pixel only after marketing consent
if (window.Noticeify.hasConsent('marketing')) {
  loadMetaPixel();
}

// React to changes in real time
window.Noticeify.onConsentChange(({ categories }) => {
  if (categories?.analytics) initGA4();
});
3

Add the footer privacy link

Required by CCPA — lets returning visitors update preferences.

<a onclick="window.Noticeify.openPreferences();return false">
  Your Privacy Choices
</a>
Shopify note

Go to Online Store → Themes → Edit code → layout/theme.liquid. Paste the embed tag immediately after <head>. GTM must load after Noticeify so Consent Mode defaults are set first.

API reference

MethodDescription
Noticeify.hasConsent('category')Returns true if the category has been granted
Noticeify.getConsent()Returns the full consent record object
Noticeify.openPreferences()Opens the preference center modal
Noticeify.onConsentChange(fn)Registers a callback fired on every consent change
Noticeify.withdrawConsent()Clears consent and re-shows the banner
